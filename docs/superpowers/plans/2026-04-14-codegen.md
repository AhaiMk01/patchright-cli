# Codegen Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `codegen` command that records user interactions and emits a shell script of patchright-cli commands.

**Architecture:** `codegen` initializes an empty list on the session. In `handle_command`, after each successful user-facing action (goto, click, fill, etc.), if recording is active, the raw command line is reconstructed and appended. `codegen-stop` writes the accumulated commands to a `.sh` file in `.patchright-cli/`.

**Tech Stack:** Python 3.10+, pytest, pytest-asyncio

```python
_CODEGEN_RECORDABLE = {
    "goto", "go-back", "go-forward", "reload",
    "click", "dblclick", "fill", "type", "hover", "select", "check", "uncheck", "drag",
    "press", "keydown", "keyup",
    "mousemove", "mousedown", "mouseup", "mousewheel",
    "scroll", "scroll-to", "wait", "wait-for",
    "tab-new", "tab-close", "tab-select",
    "upload", "resize",
    "cookie-set", "cookie-delete", "cookie-clear",
    "localstorage-set", "localstorage-delete", "localstorage-clear",
    "sessionstorage-set", "sessionstorage-delete", "sessionstorage-clear",
    "state-load",
    "route", "unroute", "network-state-set",
    "dialog-accept", "dialog-dismiss",
    "screenshot", "pdf",
}


@register("codegen")
async def cmd_codegen(session: Session, page, args: list, options: dict, cwd: str | None, state: DaemonState) -> dict:
    session._codegen = []
    return {"success": True, "output": "Recording started. Run 'codegen-stop' to save the script."}


@register("codegen-stop")
async def cmd_codegen_stop(session: Session, page, args: list, options: dict, cwd: str | None, state: DaemonState) -> dict:
    if session._codegen is None:
        return {"success": False, "output": "No recording in progress. Run 'codegen' first."}
    lines = ["#!/usr/bin/env bash", "set -e", ""] + session._codegen
    script = "\n".join(lines) + "\n"
    session._codegen = None

    base = Path(cwd) if cwd else Path.cwd()
    snap_dir = base / ".patchright-cli"
    snap_dir.mkdir(parents=True, exist_ok=True)
    ts = int(time.time() * 1000)
    filepath = args[0] if args else str(snap_dir / f"script-{ts}.sh")
    Path(filepath).parent.mkdir(parents=True, exist_ok=True)
    Path(filepath).write_text(script, encoding="utf-8")
    return {"success": True, "output": f"Saved {len(lines) - 3} command(s) to {filepath}"}
```

- [ ] **Step 5: Wire recording into handle_command**

In `handle_command`, after `return await handler(...)` and before the exception catch, add recording logic. Since `handle_command` returns a dict, we need to capture it:

Find:
```python
        handler = COMMAND_HANDLERS.get(cmd)
        if handler is None:
            return {"success": False, "output": f"Unknown command: {cmd}"}

        return await handler(session, page, args, options, cwd, state)
```

Replace with:
```python
        handler = COMMAND_HANDLERS.get(cmd)
        if handler is None:
            return {"success": False, "output": f"Unknown command: {cmd}"}

        result = await handler(session, page, args, options, cwd, state)
        if result.get("success") and session._codegen is not None and cmd in _CODEGEN_RECORDABLE:
            args_quoted = [f'"{a.replace(chr(34), chr(92) + chr(34))}"' if " " in a or '"' in a else a for a in args]
            opt_parts = []
            for k, v in options.items():
                if k in ("session", "headless", "persistent", "profile", "proxy"):
                    continue
                if v is True:
                    opt_parts.append(f"--{k}")
                elif isinstance(v, str):
                    opt_parts.append(f"--{k}={v}")
            cmd_line = " ".join(["patchright-cli"] + opt_parts + [cmd] + args_quoted)
            session._codegen.append(cmd_line)
        return result
```

Wait — we also need to record global options like `-s=session`, `--proxy`, etc. But those are per-session launch options and don't make sense inside an ongoing recording because the script should start with `open`. 

Actually, a better approach is to record the raw message as a command line. The reconstruction above is good enough for replay. Let's keep it simple.

Also, we should record the `open` command when recording starts. In `cmd_codegen`, add the current page URL if available:

```python
@register("codegen")
async def cmd_codegen(session: Session, page, args: list, options: dict, cwd: str | None, state: DaemonState) -> dict:
    lines = ["#!/usr/bin/env bash", "set -e", ""]
    if page:
        url = page.url
        lines.append(f'patchright-cli open "{url}"')
    session._codegen = lines[3:]  # exclude header lines; they'll be re-added on stop
    return {"success": True, "output": "Recording started. Run 'codegen-stop' to save the script."}
```
